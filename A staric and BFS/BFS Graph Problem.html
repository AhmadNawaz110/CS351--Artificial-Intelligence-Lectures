Introduction to the Problem

Problem Statement

Find the shortest path from Start (S) to Goal (G) in the navigation graph using Breadth-First Search (BFS).

Graph Structure

8 nodes: S, A, B, C, D, E, F, G

Edges represent legal transitions between states

Why BFS?

BFS guarantees the shortest path in unweighted graphs by exploring all nodes at the present depth before moving deeper.
